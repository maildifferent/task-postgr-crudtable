Test scenario:
- Press "Run report" button. Result: report is formed.
- Press "Create" button. Result: empty line is created.
- Enter some text into fields "Name", "Position", "Skills", "Comment" of the new empty line. Press "Commit" button. Result: new line is saved into database.
- Double click on cell "Name" in the created line. Change name. Press Enter. Press "Commit" button. Result: changes are saved into database.
- Select created line by pressing "_check" cell. Press "Delete" button. Press "Commit" button. Result: selected line is deleted from database.
- Select "name" option in the "Filter" section. Write *a* in the filter textbox. Press button "Set filter". Press button "Run report". Result: filter is set. Report is formed respecting corresponding filter.
- Using mouse select some cells area. Press Ctrl + C. Result: data is copied. You can paste it for example in Excel.
- Also it is possible to paste data into cells using Ctrl + V keys.
